  • A Navigation and Software Control System for Autonomous Mobile Robots in Subterranean Environments

    12/203,082

    The invention is a software control system. It is composed of a series of software modules that transform 3D range data into control signals that steer the robot. The 3D range data is produced via a rotating 2D LIDAR sensor. Consequently, the software processing that enables navigation is tailored to the collection and interpretation of the rotating sensor. In addition to navigation software modules, this robot has a unique software monitor that detects and recovers from faults. Navigation modules are observed and characterized while the robot is under normal operation. When the robot experiences an aberrant state of operation, the system recognizes the error and calls upon a recovery mechanism for handling these error states. Although many such recovery systems exist in software systems, this one is unique in its ability to recognize and recover from errors autonomously.

  • Software Architecture for high-speed traversal of prescribed routes

    11/761,362

    Our navigation architecture was designed with the infrastructure to support high-speed navigation while being robust to sensor failures and adaptable enough to support a rapid, relatively unstructured development process. These design goals led to a path-centric navigation architecture, built around a set of well defined, rigid data interfaces. In our path-centric architecture, the fundamental command action is to execute a path. This differs from a majority of autonomous navigation architectures which use an arc as the fundamental action.
